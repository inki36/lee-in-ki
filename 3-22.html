웹의 역사

    웹은 world web dml 줄인말이다 인터넷의 효시로는 미국 국방성에서 aparnet이다.웹브라우저는 사용자에게 편리한 인터페이스를 제공하였고 인터넷을 대중화하는데 많은 기여를 하였다

html의 기초

    html이란 hyper text markup language 의 줄인 말로서 하이퍼텍스르를 문서를 구성하기 위한 언어이다. 최근에는 새로운 버전인 html5가 각광받고있다

css의 기초

css란 cascading style sheets의 줄인말이며 웹문서의 스타일을 지정할떄 사용한다 문서 전체의 일관성을 유지할수 있어 스타일 변경에 용이하다
